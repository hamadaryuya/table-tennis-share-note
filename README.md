# :bulb:課題解決  
従来存在する卓球ノートをWeb上で使用できるようにしました。ポイントは**共有してアドバイスをお互い出来ること**です！  
個人が見るだけでは無く、人の投稿にコメント機能を搭載することで、思考力、技術力を育てます！
